## Using the Templex CLI

### List of CLI Commands

#### build

Builds the project.

```shell
yarn start build
```

#### watch

Builds the project and watches for file changes in the content, layout
and static directories. Does not watch file renaming or moving.

```shell
yarn start watch
```
